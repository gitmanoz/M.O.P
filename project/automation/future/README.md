# future/ — Engines e specs futuras

Esta pasta é reservada para documentação de Engines e capacidades ainda não priorizadas no núcleo do roadmap ativo.

## Uso

- Colocar aqui specs preliminares (ex.: `SEO_ENGINE.md`, `DEPLOY_ENGINE.md`) antes da promoção ao roadmap oficial.
- Ao aprovar uma Engine, referenciá-la em `AUTOMATION_ROADMAP.md` e, se necessário, em `BUILD_SYSTEM.md`.
- Não colocar scripts ou implementações nesta pasta nesta fase.

## Estado

Vazia de specs nomeadas na criação da arquitetura MOBS **0.1.1** — apenas reservada para crescimento.
