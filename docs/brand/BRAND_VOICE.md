## 1. Personalidade da marca

### 1.1 Quem somos

A **Missão OAB** é um **parceiro de estudo** — não uma startup genérica, não uma plataforma de entretenimento, não um cursinho tradicional.

Somos a interseção de:

```
        ORGANIZAÇÃO
            │
            │
TECNOLOGIA ─┼─ CONFIANÇA
            │
            │
        DISCIPLINA
```

### 1.2 Arquetipo de marca

| Arquetipo | Manifestação | Exemplo |
|---|---|---|
| **O Mentor** | Guia experiente que orienta sem patronizar | "Sua missão de hoje: 10 questões de Penal." |
| **O Organizador** | Traz clareza ao caos | Cronograma, trilhas, missões |
| **O Aliado** | Está ao lado do aluno, não acima | "Vamos juntos até sua aprovação." |

### 1.3 O que NÃO somos

| Não somos | Por quê |
|---|---|
| Animador / Entertainer | OAB exige seriedade |
| Vendedor agressivo | Confiança > conversão forçada |
| Guru / Influencer | Sem promessas vazias |
| Instituição distante | Somos acessíveis e humanos |
| Startup "disruptiva" | Sem jargão de pitch deck |

### 1.4 Valores da marca

| Valor | Significado | Como se manifesta |
|---|---|---|
| **Confiança** | O aluno pode confiar no conteúdo e na plataforma | Precisão jurídica, transparência |
| **Organização** | Tudo tem lugar, tudo tem ordem | Cronograma, trilhas, estrutura |
| **Tecnologia** | Ferramentas modernas a serviço do estudo | IA, analytics, plataforma |
| **Simplicidade** | Menos é mais | UI limpa, copy direta |
| **Evolução** | Melhoramos continuamente | Atualizações, feedback, iteração |
| **Disciplina** | Consistência gera resultados | Streak, missões, rotina |

### 1.5 Sensação desejada

> A sensação desejada **não é euforia**. É **confiança calma**.

O aluno deve sentir:

1. **Segurança** — "Estou no lugar certo."
2. **Clareza** — "Sei o que fazer agora."
3. **Motivação contida** — "Minha aprovação é possível."
4. **Respeito** — "Este produto leva minha jornada a sério."

## 2. Tom de voz

### 2.1 Espectro de tom

```
Formal ──────────●────────── Informal
                 ↑
            Semi-formal
         (nossa posição)

Distante ──────────●────────── Íntimo
                    ↑
               Próximo
            (nossa posição)

Sério ────────────●────────── Lúdico
                   ↑
              Profissional
           com leveza
            (nossa posição)

Técnico ──────────●────────── Simples
                   ↑
              Acessível
           com rigor
            (nossa posição)
```

### 2.2 Atributos de tom

| Atributo | Somos | Não somos |
|---|---|---|
| **Diretos** | "Entre na comunidade gratuitamente." | "Não perca essa oportunidade incrível!!!" |
| **Encorajadores** | "Sua aprovação começa hoje." | "Você VAI passar, garantido!" |
| **Profissionais** | "Materiais organizados por disciplina." | "Material top demais 🔥" |
| **Humanos** | "Junte-se a centenas de estudantes." | Linguagem robótica ou corporativa |
| **Confiantes** | "Preparação completa para o Exame de Ordem." | "Talvez a gente possa te ajudar..." |
| **Respeitosos** | "Sua jornada rumo à aprovação." | "Você precisa disso urgente!" |

### 2.3 Tom por contexto

| Contexto | Tom | Exemplo |
|---|---|---|
| Landing / Marketing | Confiante, acolhedor | "Sua aprovação começa hoje." |
| Onboarding | Guia, simples | "Vamos configurar seu perfil." |
| Estudo (questões) | Neutro, funcional | "Questão 5 de 10 — Direito Penal." |
| Feedback (acerto) | Positivo, contido | "Resposta correta. +10 XP." |
| Feedback (erro) | Didático, encorajador | "Resposta incorreta. Veja o comentário." |
| Gamificação | Reconhecimento, profissional | "Nível 10 — Analista." |
| Premium | Valor claro, sem pressão | "Desbloqueie todo seu potencial." |
| Erro | Empático, solucionador | "Algo deu errado. Tente novamente." |
| Suporte | Humano, paciente | "Entendemos sua dúvida. Veja como resolver." |
| Email | Pessoal, informativo | "Olá, Ana. Seu resumo semanal." |

### 2.4 Nível de formalidade

**Semi-formal** — equilíbrio entre proximidade e autoridade acadêmica.

| Situação | Formalidade | Exemplo |
|---|---|---|
| Headline marketing | Semi-formal | "Sua aprovação começa hoje." |
| Instrução de estudo | Semi-formal | "Resolva 10 questões de Penal." |
| Comunicação B2B | Formal | "Prezados parceiros," |
| WhatsApp (grupo) | Informal controlado | "Boa noite! Questão do dia:" |
| Termos legais | Formal | "O usuário concorda em..." |
| Notificação push | Informal | "Sua missão de hoje te espera." |

## 5. Palavras que nunca utilizamos

### 5.1 Lista proibida

| Palavra/Expressão | Por quê | Alternativa |
|---|---|---|
| garantido / garantia de aprovação | Antiético, ilegal, falso | "preparação completa" |
| milagre / milagroso | Sensacionalismo | "método organizado" |
| hack / truque / segredo | Clickbait | "estratégia de estudo" |
| fórmula mágica | Desonestidade | "método comprovado" |
| imperdível / última chance | Urgência falsa | "comece agora" |
| grátis total / de graça | Informal demais | "gratuitamente" / "sem custo" |
| top / demais / sensacional | Informal demais | "completo" / "organizado" |
| 🔥 / 🚀 / 💯 (excesso) | Informal demais para UI | Ícones profissionais |
| barato / preço baixo | Desvaloriza | "acessível" / "investimento" |
| fácil / simples demais | Minimiza desafio OAB | "organizado" / "estruturado" |
| reprovado / fracasso | Negativo, desmotivador | "próxima tentativa" / "retomar" |
| concorrente / outro curso | Foco interno | Não mencionar |
| melhor do mercado | Arrogância | "preparação completa" |
| revolucionário / disruptivo | Jargão startup | "inteligente" / "moderno" |
| guru / mestre / ninja | Clichê marketing | "parceiro de estudo" |
| hackathon / gamificação extrema | Infantilização | "missões" / "desafios" |

### 5.2 Padrões proibidos

| Padrão | Exemplo proibido | Correto |
|---|---|---|
| CAPS LOCK | "VOCÊ VAI PASSAR!" | "Sua aprovação começa hoje." |
| Múltiplas exclamações | "Incrível!!!" | "Resultado registrado." |
| Emoji excessivo | "Bora estudar 🔥🔥🔥" | "Sua missão de hoje te espera." |
| Urgência falsa | "Últimas 3 vagas!" | "Entre gratuitamente." |
| Promessa numérica | "90% de aprovação!" | "Preparação completa." |
| Linguagem de vendas | "Compre agora e ganhe!" | "Conheça o Premium." |

## 6. Como escrever para alunos

### 6.1 Princípios

| Princípio | Aplicação |
|---|---|
| Tratar como adulto | Alunos são bacharéis/profissionais |
| Respeitar o esforço | Reconhecer dedicação, não minimizar |
| Ser útil | Toda mensagem deve ajudar a estudar |
| Ser honesto | Não prometer o que não podemos entregar |
| Ser presente | Acompanhar a jornada, não desaparecer |

### 6.2 Pessoa gramatical

| Contexto | Pessoa | Exemplo |
|---|---|---|
| Marketing | 2ª pessoa singular (você) | "Organize seus estudos." |
| In-app | 2ª pessoa singular | "Sua missão de hoje:" |
| Email | 2ª pessoa + nome | "Olá, Ana. Seu resumo:" |
| WhatsApp | 2ª pessoa + tom leve | "Boa noite! Questão do dia:" |
| FAQ | 3ª pessoa ou impessoal | "A comunidade é indicada para..." |

### 6.3 Exemplos por situação

| Situação | ❌ Errado | ✅ Correto |
|---|---|---|
| Boas-vindas | "E aí! Bem-vindo ao melhor curso de OAB!" | "Bem-vindo à Missão OAB. Sua preparação começa agora." |
| Motivação | "Você é incrível! Vai arrasar na prova!" | "14 dias consecutivos de estudo. Continue assim." |
| Erro em questão | "Errou feio! Estude mais!" | "Resposta incorreta. Veja o comentário para entender." |
| Simulado ruim | "Poxa, nota baixa..." | "Seu resultado: 52%. Foco em Processo Civil (38%)." |
| Simulado bom | "ARRASOU! Nota TOP!" | "Seu melhor resultado: 78%. Evolução de 12%." |
| Premium | "Compre JÁ ou vai reprovar!" | "Desbloqueie questões ilimitadas e IA tutor." |
| Streak perdido | "Você falhou no streak!" | "Seu streak reiniciou. Comece um novo hoje." |

### 6.4 Tratamento de gênero

| Regra | Aplicação |
|---|---|
| Neutro quando possível | "Bem-vindo(a)" ou "Bem-vindo à Missão OAB" |
| Personalizado com nome | "Olá, Ana." / "Olá, Rafael." |
| Sem assumir gênero | Evitar "cara", "mano", "querida" |
| Conquistas neutras | "Conquista desbloqueada" (não "Conquista desbloqueada, campeão!") |

## 7. Como escrever para parceiros

### 7.1 Tom

| Atributo | Descrição |
|---|---|
| Formalidade | Formal a semi-formal |
| Pessoa | 2ª pessoa plural (vocês) ou 3ª pessoa |
| Foco | Valor mútuo, credibilidade, números |
| Evitar | Informalidade, promessas exageradas |

### 7.2 Vocabulário B2B

| Usar | Evitar |
|---|---|
| parceria | parceria incrível |
| colaboração | negócio |
| alinhamento | sinergia |
| curadoria | qualquer conteúdo |
| comunidade ativa | base de usuários |
| plataforma de preparação | app / site |
| receita compartilhada | ganhar dinheiro fácil |

### 7.3 Exemplos

**Email de proposta de parceria:**
```
Assunto: Proposta de parceria — Missão OAB × {empresa}

Prezados,

A Missão OAB é uma plataforma de preparação inteligente para o Exame de Ordem,
com {numero} alunos ativos e crescimento de {percentual}% ao mês.

Identificamos oportunidade de colaboração em {area}, alinhada ao nosso
compromisso com qualidade e relevância para o aluno.

Gostaríamos de agendar uma conversa para apresentar nossa proposta.

Atenciosamente,
{nome}
Missão OAB
```

**Descrição no marketplace:**
```
{Parceiro} — {titulo_curso}

{descricao_2_linhas}

✓ Alinhado ao edital OAB
✓ Avaliação: {nota}/5.0 ({avaliacoes} avaliações)
✓ Por: {nome_parceiro}, {credencial}

[ Ver curso ]
```

## 8. Como escrever para patrocinadores

### 8.1 Tom

| Atributo | Descrição |
|---|---|
| Formalidade | Formal |
| Pessoa | 3ª pessoa institucional |
| Foco | Audiência qualificada, brand safety, ROI |
| Evitar | Promessas de conversão direta, linguagem de vendas |

### 8.2 Vocabulário patrocínio

| Usar | Evitar |
|---|---|
| audiência qualificada | massa de usuários |
| brand safety | qualquer anúncio |
| conteúdo patrocinado | propaganda |
| associação de marca | logo em todo lugar |
| público de alta intenção | leads baratos |
| preparação profissional | cursinho |

### 8.3 Exemplos

**Proposta de patrocínio:**
```
Assunto: Oportunidade de patrocínio — Missão OAB

Prezados,

A Missão OAB alcança mensalmente {MAU} estudantes de Direito em fase
de preparação para o Exame de Ordem — um público de alta intenção
e decisão de compra.

Oferecemos oportunidades de associação de marca em:
• Simulados nacionais (alcance: {numero} participantes)
• Conteúdo editorial ("Apresentado por {marca}")
• Eventos online (webinars, maratonas)

Nosso compromisso: brand safety total. Conteúdo patrocinado é
claramente identificado e nunca interfere na experiência de estudo.

Segue mídia kit em anexo.

Atenciosamente,
{nome}
Missão OAB — Parcerias
```

**Label de conteúdo patrocinado (in-app):**
```
Apresentado por {marca}
```

## 9. Exemplos práticos

### 9.1 Mesma mensagem, tons diferentes

**Contexto: Aluno completou 100 questões**

| Audiência | Copy |
|---|---|
| Aluno (in-app) | Conquista desbloqueada: Centena. 100 questões resolvidas. |
| Aluno (push) | 100 questões resolvidas. Conquista desbloqueada. |
| Aluno (email) | Olá, Ana. Você atingiu 100 questões resolvidas. Sua dedicação está gerando resultados. |
| Parceiro (report) | Usuários ativos resolveram média de 100+ questões/mês. |
| Patrocinador (deck) | Engajamento médio: 100+ interações de estudo por usuário ativo/mês. |

### 9.2 Antes e depois (revisão de copy)

| Antes (❌) | Depois (✅) | Motivo |
|---|---|---|
| "Curso TOP de OAB!" | "Preparação completa para o Exame de Ordem." | Profissionalismo |
| "Garantimos sua aprovação!" | "Sua aprovação começa hoje." | Honestidade |
| "Corre! Últimas vagas!" | "Entre gratuitamente na comunidade." | Sem urgência falsa |
| "Você errou! Tente de novo." | "Resposta incorreta. Veja o comentário." | Respeito |
| "Compre Premium JÁ!" | "Conheça o Premium." | Sem pressão |
| "Parabéns, campeão! 🏆🎉" | "Conquista desbloqueada: Mês Completo." | Tom profissional |

## 10. Checklist de voz

### Antes de publicar qualquer comunicação

```
┌────────────────────────────────────────────────────────────┐
│  CHECKLIST DE VOZ — MISSÃO OAB                             │
├────────────────────────────────────────────────────────────┤
│  IDENTIDADE                                                │
│  [ ] Soa como a Missão OAB (não genérico)?                 │
│  [ ] Transmite confiança e organização?                    │
│  [ ] Respeita o aluno como adulto/profissional?            │
│                                                            │
│  TOM                                                       │
│  [ ] Semi-formal (não muito informal, não muito distante)? │
│  [ ] Encorajador sem ser exagerado?                        │
│  [ ] Direto e claro?                                       │
│                                                            │
│  VOCABULÁRIO                                               │
│  [ ] Usa termos oficiais (comunidade, plataforma, missão)? │
│  [ ] Evita palavras proibidas (garantido, hack, milagre)?  │
│  [ ] Sem CAPS LOCK ou exclamações excessivas?              │
│                                                            │
│  AUDIÊNCIA                                                 │
│  [ ] Tom adequado ao destinatário (aluno/parceiro/sponsor)?│
│  [ ] Pessoa gramatical correta?                            │
│  [ ] Gênero neutro ou personalizado?                       │
│                                                            │
│  AÇÃO                                                      │
│  [ ] Toda mensagem ajuda o aluno (estudar/retornar/decidir)?│
│  [ ] CTA claro (se aplicável)?                             │
└────────────────────────────────────────────────────────────┘
```
