## 1. Princípios de copywriting

### Regras fundamentais

| # | Regra | Exemplo |
|---|---|---|
| 1 | **Clareza acima de criatividade** | "Entre na comunidade gratuitamente" > "Desbrave sua jornada" |
| 2 | **Benefício, não feature** | "Saiba o que estudar hoje" > "Temos cronograma" |
| 3 | **Ação orientada** | Verbos no imperativo ou infinitivo: "Comece", "Resolva", "Entre" |
| 4 | **Segunda pessoa** | "Você" — fala diretamente com o aluno |
| 5 | **Sem hype** | Nunca "garantido", "milagre", "segredo", "hack" |
| 6 | **Prova concreta** | "Centenas de estudantes" > "Milhares de alunos satisfeitos" |
| 7 | **Breve** | Headlines ≤ 8 palavras; parágrafos ≤ 3 linhas |
| 8 | **Consistência** | Mesmo termo para mesma coisa em todo o produto |

### Fórmulas de copy

| Fórmula | Estrutura | Exemplo |
|---|---|---|
| **AIDA** | Atenção → Interesse → Desejo → Ação | Hero landing |
| **PAS** | Problema → Agitação → Solução | Seção benefícios |
| **BAB** | Before → After → Bridge | Depoimentos |
| **4U** | Useful, Urgent, Unique, Ultra-specific | CTAs |

## 3. Botões e CTAs

### 3.1 Hierarquia de CTAs

| Nível | Estilo | Uso | Exemplo |
|---|---|---|---|
| **Primário** | Botão sólido azul | Ação principal da tela | "Quero Entrar na Missão" |
| **Secundário** | Botão outline | Ação alternativa | "Ver Como Funciona" |
| **Terciário** | Link texto | Navegação | "Saiba mais" |
| **Destrutivo** | Botão vermelho | Ações irreversíveis | "Excluir conta" |

### 3.2 CTAs por tela (futuro)

| Tela | CTA primário | CTA secundário |
|---|---|---|
| Landing | Quero Entrar na Missão | — |
| Cadastro | Criar conta | Continuar com Google |
| Login | Entrar | Esqueci minha senha |
| Onboarding | Próximo / Começar | Pular |
| Dashboard | Iniciar estudo | Ver cronograma |
| Questão | Confirmar resposta | Pular questão |
| Simulado | Finalizar simulado | Pausar |
| Missão | Iniciar missão | Ver detalhes |
| Premium | Assinar Premium | Continuar gratuito |
| Perfil | Salvar alterações | — |

### 3.3 Regras de botões

| Regra | Detalhe |
|---|---|
| Verbo de ação | Sempre começar com verbo |
| ≤ 4 palavras | "Quero Entrar na Missão" (4 palavras) ✓ |
| Sem emoji na UI | Emojis só em WhatsApp/notificações, nunca em botões |
| Consistência | Mesmo CTA = mesma ação em todo o produto |
| Loading state | "Entrando..." / "Salvando..." / "Carregando..." |

## 8. Mensagens de erro

### 8.1 Princípios

| Princípio | Aplicação |
|---|---|
| Humano | "Algo deu errado" > "Error 500" |
| Ação clara | Sempre dizer o que fazer |
| Sem culpa | Nunca "Você errou" |
| Breve | 1–2 frases |

### 8.2 Templates

| Contexto | Título | Mensagem | Ação |
|---|---|---|---|
| Login falhou | Email ou senha incorretos | Verifique seus dados e tente novamente. | Tentar novamente |
| Email já cadastrado | Email já em uso | Este email já possui uma conta. Faça login ou recupere sua senha. | Fazer login |
| Senha fraca | Senha insuficiente | Use pelo menos 8 caracteres, incluindo letras e números. | — |
| Sem conexão | Sem conexão com a internet | Verifique sua conexão e tente novamente. | Tentar novamente |
| Erro genérico | Algo deu errado | Não foi possível completar a ação. Tente novamente em instantes. | Tentar novamente |
| Sessão expirada | Sessão expirada | Faça login novamente para continuar. | Fazer login |
| Limite atingido | Limite diário atingido | Você resolveu 30 questões hoje. Amanhã o limite reinicia, ou assine Premium para acesso ilimitado. | Ver Premium |
| Pagamento falhou | Pagamento não processado | Verifique os dados do cartão ou tente outro método. | Tentar novamente |
| Página não encontrada | Página não encontrada | O endereço que você acessou não existe. Volte ao início. | Ir para início |

## 11. Tom emocional por momento

### 11.1 Mapa emocional

| Momento | Emoção do aluno | Tom da copy | Exemplo |
|---|---|---|---|
| Descoberta (ad) | Curiosidade cética | Confiante, direto | "Organize seus estudos para a OAB." |
| Landing | Avaliação | Profissional, acolhedor | "Sua aprovação começa hoje." |
| Cadastro | Expectativa | Simples, rápido | "Crie sua conta em 1 minuto." |
| 1º estudo | Concentração | Neutro, funcional | "Questão 1 de 5 — Ética Profissional" |
| 1º acerto | Satisfação | Contido, positivo | "Resposta correta. +10 XP." |
| 1º erro | Frustração | Encorajador, didático | "Resposta incorreta. Veja o comentário." |
| Streak 7 dias | Orgulho | Reconhecimento | "7 dias consecutivos de estudo." |
| Simulado ruim | Desânimo | Empático, construtivo | "Seu resultado: 52%. Foco em {disciplina}." |
| Simulado bom | Confiança | Reconhecimento | "Seu melhor resultado: 78%. Continue." |
| Pré-prova | Ansiedade | Calmo, preparado | "Você estudou {horas}h. Confie no processo." |
| Premium | Decisão | Valor claro, sem pressão | "Desbloqueie todo seu potencial." |
| Aprovação | Euforia | Celebração contida | "Parabéns pela aprovação. Você merece." |

### 11.2 O que NUNCA escrever

| ❌ Evitar | ✅ Usar |
|---|---|
| "Você VAI passar!" | "Sua aprovação começa hoje." |
| "Não perca essa chance!" | "Entre gratuitamente." |
| "Últimas vagas!" | "Junte-se à comunidade." |
| "Método secreto" | "Método organizado" |
| "Garantia de aprovação" | "Preparação completa" |
| "Hack para passar" | "Estratégia de estudo" |
| "Material TOP 🔥" | "Material organizado por disciplina" |
| "Corre!!!" | "Comece agora." |
