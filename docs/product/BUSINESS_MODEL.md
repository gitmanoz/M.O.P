## 1. Visão financeira

### 1.1 Tese central

> **A Missão OAB será sustentável financeiramente quando a receita recorrente (MRR) cobrir os custos operacionais e o CAC for recuperado em ≤ 6 meses via LTV.**

### 1.2 Horizonte financeiro

```
2026                    2027                    2028
Q3    Q4    Q1    Q2    Q3    Q4    Q1    Q2    Q3    Q4
│     │     │     │     │     │     │     │     │     │
├─ Investimento ────────┤     │     │     │     │     │
│  (V1–V2: aquisição)   │     │     │     │     │     │
│                       ├─ Crescimento ─────────┤     │
│                       │  (V3–V4: produto)   │     │
│                       │                     ├─ Escala ──>
│                       │                     │  (V5–V6)
│  MRR: R$ 0            │  MRR: R$ 0          │  MRR: R$ 30k+
│  Burn: ~R$ 5k/mês      │  Burn: ~R$ 15k/mês  │  Break-even
```

### 1.3 Metas financeiras por fase

| Fase | Período | MRR alvo | Burn mensal | Runway mínimo |
|---|---|---|---|---|
| Validação (V1) | Q3–Q4 2026 | R$ 0 | R$ 5.000 | 12 meses |
| Produto (V2–V3) | Q4 2026–Q2 2027 | R$ 0 | R$ 15.000 | 12 meses |
| Monetização (V5) | Q4 2027 | R$ 30.000 | R$ 25.000 | 6 meses |
| Escala (V6+) | 2028 | R$ 100.000 | Break-even | Autossustentável |

## 2. Princípios de monetização

### 2.1 Princípios inegociáveis

```
┌────────────────────────────────────────────────────────────┐
│           PRINCÍPIOS DE MONETIZAÇÃO — MISSÃO OAB           │
├────────────────────────────────────────────────────────────┤
│                                                            │
│  1. EXPERIÊNCIA DO ALUNO > MONETIZAÇÃO                     │
│     Nunca bloquear o essencial do estudo.                  │
│                                                            │
│  2. GRATUIDADE GENEROSA                                    │
│     Free tier deve ser genuinamente útil para aprovação.   │
│                                                            │
│  3. PREMIUM = ACELERADOR, NÃO PORTEIRO                     │
│     Premium acelera e aprofunda; Free já permite aprovar.  │
│                                                            │
│  4. TRANSPARÊNCIA TOTAL                                    │
│     Preços claros, sem taxas escondidas, cancelamento fácil.│
│                                                            │
│  5. NUNCA PREJUDICAR QUEM NÃO PAGA                         │
│     Aluno Free recebe respeito e qualidade.                │
│                                                            │
│  6. MONETIZAÇÃO PROGRESSIVA                                │
│     Só monetizar após entregar valor comprovado (V5).      │
│                                                            │
│  7. DIVERSIFICAÇÃO DE RECEITA                              │
│     Não depender de uma única fonte.                       │
│                                                            │
```

### 2.2 O que NUNCA faremos

| Prática | Por quê |
|---|---|
| Paywall em questões básicas | Destrói confiança e missão |
| Anúncios invasivos | Interrompe estudo |
| Venda de dados de alunos | LGPD + ética |
| Preço abusivo | Aluno OAB já gasta muito |
| Cancelamento difícil | Dark pattern |
| Conteúdo inferior no Free | Desrespeito ao aluno |
| "Garantia de aprovação" paga | Antiético e ilegal |

## 6. Unit economics

### 6.1 CAC (Customer Acquisition Cost)

```
                    Custo total de marketing + vendas
CAC = ─────────────────────────────────────────────────
              Novos usuários adquiridos no período
```

| Canal | CAC estimado | Volume | Qualidade |
|---|---|---|---|
| Meta Ads | R$ 12–18 | Alto | Média |
| Orgânico (SEO) | R$ 2–5 | Médio | Alta |
| Indicação | R$ 5–8 | Baixo | Muito alta |
| WhatsApp (V1) | R$ 10–15 | Médio | Alta |
| **Blended CAC** | **R$ 12–15** | — | — |

### 6.2 LTV (Lifetime Value)

```
LTV = ARPU × Margem bruta × (1 / Churn mensal)
```

| Cenário | ARPU | Margem | Churn | LTV | LTV/CAC |
|---|---|---|---|---|---|
| Conservador | R$ 8 | 70% | 10%/mês | R$ 56 | 3,7× |
| Base | R$ 12 | 75% | 8%/mês | R$ 112 | 7,5× |
| Otimista | R$ 18 | 80% | 5%/mês | R$ 288 | 19× |

> **Meta: LTV/CAC ≥ 3×** (saudável para SaaS/EdTech)

### 6.3 ARPU (Average Revenue Per User)

```
                    MRR total
ARPU = ─────────────────────────
       Total de usuários ativos
```

| Fase | MAU | Assinantes Premium | MRR | ARPU |
|---|---|---|---|---|
| V5 (Q4 2027) | 1.500 | 75 (5%) | R$ 3.750 | R$ 2,50 |
| V5+6 meses | 3.000 | 210 (7%) | R$ 10.500 | R$ 3,50 |
| V6 (2028) | 8.000 | 640 (8%) | R$ 32.000 | R$ 4,00 |

### 6.4 Payback period

```
                    CAC
Payback = ─────────────────────────
          ARPU Premium × Margem
```

| Cenário | CAC | ARPU Premium | Margem | Payback |
|---|---|---|---|---|
| Base | R$ 15 | R$ 45 | 75% | 0,44 meses (~13 dias) |
| Conservador | R$ 20 | R$ 40 | 70% | 0,71 meses (~21 dias) |

> Payback rápido porque Premium converte alunos já adquiridos (custo marginal ≈ 0).
