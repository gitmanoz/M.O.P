## 1. Filosofia

### Por que gamificamos

> **Gamificação na Missão OAB não é entretenimento. É engenharia de hábitos a serviço da aprovação.**

O aluno OAB enfrenta meses de estudo solitário e repetitivo. Gamificação transforma consistência em progresso visível — sem trivializar a seriedade da preparação.

### O que gamificação NÃO é aqui

| Não é | É |
|---|---|
| Jogo | Ferramenta de disciplina |
| Diversão | Motivação estruturada |
| Competição tóxica | Competição saudável |
| Infantilização | Reconhecimento profissional |
| Distração do estudo | Incentivo ao estudo |

### Analogia

```
Gamificação Missão OAB = Academia de alto performance
                         (não parque de diversões)

  ✓ Metas claras          ✗ "Colete moedas!"
  ✓ Progresso mensurável  ✗ Avatares cartoon
  ✓ Reconhecimento real   ✗ Confetes excessivos
  ✓ Disciplina            ✗ Competição destrutiva
```

## 2. Princípios

| # | Princípio | Aplicação |
|---|---|---|
| G1 | **Estudo gera XP, não cliques** | XP só por ações de estudo verificáveis |
| G2 | **Tom profissional** | "Analista Jurídico Nível 12", não "Mestre das Questões 🎮" |
| G3 | **Progresso > Competição** | Ranking complementa, não substitui, progresso pessoal |

| G5 | **Anti-fraude by design** | Sistema detecta e penaliza farming |
| G6 | **Recompensa esforço real** | Streak de estudo, não streak de login |
| G7 | **Alinhado ao calendário OAB** | Temporadas seguem edições do exame |
| G8 | **Free participa plenamente** | Gamificação não é paywall |

## 3. XP (Experience Points)

### 3.1 Tabela de XP por ação

| Ação | XP base | Limite/dia | Condição |
|---|---|---|---|
| Resolver questão (acerto) | 10 | 300 (30 questões) | Resposta correta |
| Resolver questão (erro) | 5 | 150 (30 questões) | Resposta incorreta (participou) |
| Completar missão diária | 50 | 50 (1 missão) | 100% da missão |
| Completar missão semanal | 200 | 200 (1 missão) | 100% da missão |
| Completar simulado | 100 | 100 (1 simulado) | Finalizar 80 questões |
| Simulado: score ≥ 70% | +50 bônus | 50 | Performance |
| Simulado: score ≥ 85% | +100 bônus | 100 | Performance excepcional |
| Streak: dia completado | 15 | 15 | ≥ 1 questão no dia |
| Streak: marco 7 dias | 100 | — | Bônus único/semana |
| Streak: marco 30 dias | 500 | — | Bônus único/mês |
| Conquista desbloqueada | 25–500 | — | Varia por conquista |
| Evento especial completado | 100–300 | — | Varia por evento |
| Revisão espaçada (IA) | 20 | 60 (3 revisões) | Completar revisão sugerida |

### 3.2 XP máximo teórico/dia

```
Questões (30 acertos × 10)     = 300 XP
Missão diária                  =  50 XP
Streak diário                  =  15 XP
Simulado (base + bônus 70%)    = 150 XP
Revisão IA (3 × 20)            =  60 XP
─────────────────────────────────────
Máximo/dia                     = 575 XP
Máximo/semana (realista)       ≈ 2.500 XP
```

### 3.3 Fórmula de XP com multiplicador

```
XP_final = XP_base × multiplicador_streak × multiplicador_liga

Multiplicador streak:
  1–6 dias:   1.0×
  7–13 dias:  1.1×
  14–29 dias: 1.2×
  30+ dias:   1.3×

Multiplicador liga:
  Bronze:     1.0×
  Prata:      1.05×
  Ouro:       1.10×
  Diamante:   1.15×
  Elite:      1.20×
```

## 5. Ligas

### 5.1 Estrutura de ligas

```
                    ┌─────────────┐
                    │    ELITE    │  Top 5% — convite por performance
                    └──────▲──────┘
                           │
                    ┌──────┴──────┐
                    │  DIAMANTE   │  Top 15%
                    └──────▲──────┘
                           │
                    ┌──────┴──────┐
                    │    OURO     │  Top 30%
                    └──────▲──────┘
                           │
                    ┌──────┴──────┐
                    │    PRATA    │  Top 50%
                    └──────▲──────┘
                           │
                    ┌──────┴──────┐
                    │   BRONZE    │  Todos começam aqui
                    └─────────────┘
```

### 5.2 Regras de promoção/rebaixamento

| Liga | Promoção (top X sobem) | Rebaixamento (bottom Y descem) |
|---|---|---|
| Bronze | Top 20 sobem → Prata | — (liga inicial) |
| Prata | Top 15 sobem → Ouro | Bottom 5 descem → Bronze |
| Ouro | Top 10 sobem → Diamante | Bottom 5 descem → Prata |
| Diamante | Top 5 sobem → Elite | Bottom 5 descem → Ouro |
| Elite | — (liga máxima) | Bottom 5 descem → Diamante |

### 5.3 Ciclo de ligas

- **Duração:** 1 semana (segunda 00:00 → domingo 23:59)
- **Reset:** XP semanal zera; XP acumulado (nível) permanece
- **Grupo:** ~30 alunos por grupo de liga (matchmaking por XP similar)

## 10. Streak

### 10.1 Regras

| Regra | Detalhe |
|---|---|
| **O que conta** | Resolver ≥ 1 questão OU estudar ≥ 15 min |
| **Reset** | Meia-noite (timezone America/Sao_Paulo) |
| **Grace period** | 1 dia/semana (escudo automático) |
| **Escudo extra** | Conquista "Recuperação" após perder streak |
| **Máximo** | Sem limite (recorde pessoal) |
| **Notificação** | 20h se não estudou: "Seu streak de X dias está em risco" |

### 10.2 Fluxo de streak

```
Dia 1: Estudou ✓ → Streak: 1
Dia 2: Estudou ✓ → Streak: 2
Dia 3: Não estudou → Grace period usado → Streak: 2 (mantido)
Dia 4: Estudou ✓ → Streak: 3
Dia 5: Não estudou (grace já usado) → Streak: 0 (reset)
Dia 6: Estudou ✓ → Streak: 1 (recomeço)
```

## 15. Anti-fraude

### 15.1 Vetores de fraude

| Vetor | Descrição | Detecção |
|---|---|---|
| Click farming | Responder questões aleatoriamente rápido | Tempo < 5s/questão |
| Bot | Automação de respostas | Padrão regular de timing |
| Multi-conta | Várias contas para ranking | IP, device fingerprint |
| Login sem estudo | Login diário sem questões | Streak sem XP de questão |
| Compartilhamento de conta | Conta usada por 2+ pessoas | Padrões de horário/dispositivo |

### 15.2 Sistema de detecção

```
┌─────────────────────────────────────────────────────────┐
│  PIPELINE ANTI-FRAUDE                                   │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  Ação do aluno                                          │
│       │                                                 │
│       ▼                                                 │
│  ┌─────────────┐                                        │
│  │  Validação  │  Tempo mínimo? Padrão normal?          │
│  │  em tempo   │  Device conhecido?                     │
│  │  real       │                                        │
│  └──────┬──────┘                                        │
│         │                                               │
│    ┌────┴────┐                                          │
│    ▼         ▼                                          │
│  OK      Suspeito                                       │
│    │         │                                          │
│    ▼         ▼                                          │
│  XP      Flag + review                                  │
│  atribuído  │                                           │
│             ▼                                           │
│        ┌─────────┐                                      │
│        │ Penalidade│                                     │
│        │ - XP     │                                      │
│        │ - Streak │                                      │
│        │ - Ban    │ (casos graves)                       │
│        └─────────┘                                      │
└─────────────────────────────────────────────────────────┘
```

### 15.3 Penalidades

| Infração | 1ª vez | 2ª vez | 3ª vez |
|---|---|---|---|
| Resposta rápida demais (< 3s) | XP não contabilizado | Aviso | Suspensão 24h |
| Padrão de bot detectado | Aviso + XP removido | Suspensão 7 dias | Ban permanente |
| Multi-conta | Ban secundária | Ban primária | — |
| Exploit de bug | XP removido + report | Suspensão | Ban |

### 15.4 Apelação

- Aluno pode contestar penalidade via suporte
- Review manual em ≤ 48h
- Histórico de decisões documentado
