## 1. Visão geral da jornada

A jornada do aluno Missão OAB é dividida em **11 estágios**, do primeiro contato com anúncio até a conversão Premium e retenção de longo prazo.

### Princípio orientador

> Cada estágio deve entregar **valor imediato** antes de pedir qualquer compromisso adicional.

### Persona de referência

**Ana, 24 anos** — recém-formada, 1ª tentativa OAB, estuda 1,5h/dia após trabalho.

## Estágio 1 — Meta Ads

> **Versão:** V1  
> **Objetivo:** Captar atenção e gerar clique  
> **Métrica chave:** CTR ≥ 1,5%

### Fluxo

```
Feed Instagram/Facebook
        │
        ▼
┌───────────────────┐
│  Criativo (imagem  │
│  ou vídeo 15–30s) │
│                   │
│  Headline:        │
│  "Organize seus   │
│   estudos para    │
│   a OAB"          │
│                   │
│  [ Saiba mais ]   │
└─────────┬─────────┘
          │ clique
          ▼
    Landing Page
```

### Touchpoints

| Canal | Formato | Frequência |
|---|---|---|
| Instagram Feed | Carrossel / imagem | Contínuo |
| Instagram Stories | Vídeo 15s | Contínuo |
| Facebook Feed | Imagem / vídeo | Contínuo |
| Remarketing | Dinâmico | Visitantes landing |

### O que o aluno pensa

| Momento | Pensamento | Emoção |
|---|---|---|
| Vê o anúncio | "Será que isso é diferente do que já tentei?" | Curiosidade cética |
| Lê headline | "Organização... é exatamente o que me falta" | Identificação |
| Vê prova social | "Centenas de estudantes... não estou sozinha" | Alívio |
| Decide clicar | "Vou dar uma olhada, não custa nada" | Esperança cautelosa |

### Pontos de dor

| Dor | Descrição | Severidade |
|---|---|---|
| Ceticismo | "Mais um curso/grupo que promete e não entrega" | Alta |
| Saturação | Já viu muitos anúncios de preparação | Média |
| Falta de diferenciação | Anúncio parece genérico | Média |

### Momentos de encantamento

| Momento | O que encanta | Como |
|---|---|---|
| Headline específica | Fala de "organização", não "aprovação garantida" | Copy honesta |
| Visual limpo | Design profissional, não amador | Identidade visual |
| Gratuidade clara | "Comunidade gratuita" visível | Transparência |

### Oportunidades

| # | Oportunidade | Impacto | Esforço |
|---|---|---|---|
| O1 | Vídeo testimonial de aluno real | Alto | Médio |
| O2 | Criativo com cronograma visual | Alto | Baixo |
| O3 | A/B test: "organização" vs "comunidade" | Médio | Baixo |
| O4 | Lookalike audience de leads convertidos | Alto | Baixo |

### Métricas

| Métrica | Meta | Alarme |
|---|---|---|
| CTR | ≥ 1,5% | < 0,8% |
| CPC | < R$ 2,00 | > R$ 4,00 |
| CPM | < R$ 25 | > R$ 40 |

> **Objetivo:** Converter visitante em lead (clique WhatsApp)  
> **Métrica chave:** Taxa conversão CTA ≥ 8%

```

        │
        ▼ (scroll)

│  Cronograma · PDFs · Questões │
│  Simulados · Resumos · Atual. │
└───────────────────────────────┘
        │
        ▼
┌───────────────────────────────┐
│  COMO FUNCIONA (4 passos)     │
└───────────────────────────────┘
        │
        ▼
┌───────────────────────────────┐
│  FAQ (4 perguntas)            │
└───────────────────────────────┘
        │
        ▼
┌───────────────────────────────┐
│  CTA FINAL                    │
│  [ Quero Entrar na Missão ]   │──────> WhatsApp
└───────────────────────────────┘
```

### O que o aluno pensa

| Seção | Pensamento | Emoção |
|---|---|---|
| Hero | "Parece sério. Não é golpe." | Confiança inicial |
| Benefícios | "Cronograma + questões + simulados... completo" | Interesse |
| Como Funciona | "Só 4 passos, parece simples" | Alívio |
| FAQ | "É gratuito? Sim. Para quem? Para mim." | Validação |
| CTA Final | "Vou entrar. Não tenho nada a perder." | Decisão |

### Pontos de dor

| Dor | Descrição | Severidade | Mitigação |
|---|---|---|---|
| CTA sem link real | Botão leva a `#` | Crítica | Configurar URL WhatsApp |
| Falta de prova social | "Centenas" sem evidência | Média | Depoimentos, contador |
| Mobile lento | Page load > 3s | Alta | Otimizar performance |
| Informação insuficiente | Quer saber quem está por trás | Média | Seção "Quem somos" (V1.1) |

### Momentos de encantamento

| Momento | O que encanta |
|---|---|
| Design limpo e profissional | "Isso parece um produto de verdade" |
| FAQ responde objeções | "Gratuito, para formandos, sem app" |
| Timeline clara | "Sei exatamente o que vai acontecer" |
| Velocidade de carregamento | "Rápido, funciona no celular" |

### Oportunidades

| # | Oportunidade | Impacto |
|---|---|---|
| O5 | Adicionar contador de membros ao vivo | Alto |
| O6 | Vídeo de 30s no hero | Alto |
| O7 | Depoimentos com foto e nome | Alto |
| O8 | Exit-intent popup com benefício extra | Médio |

### Métricas

| Métrica | Meta | Alarme |
|---|---|---|
| Taxa conversão CTA | ≥ 8% | < 4% |
| Tempo na página | ≥ 45s | < 20s |
| Bounce rate | < 55% | > 70% |
| Scroll depth (CTA final) | ≥ 40% | < 20% |

## Estágio 3 — Grupo WhatsApp

> **Versão:** V1  
> **Objetivo:** Entregar valor imediato e construir confiança  
> **Métrica chave:** Retenção 7 dias ≥ 60%

### Fluxo

```
Clique CTA → Abre WhatsApp → Entra no grupo
                                    │
                    ┌───────────────┼───────────────┐
                    ▼               ▼               ▼
              Mensagem         Conteúdo        Interação
              boas-vindas      diário          com membros
              (automática      (PDF, questão,  (dúvidas,
               ou manual)       cronograma)      motivação)
```

### Calendário editorial sugerido

| Dia | Conteúdo | Formato |
|---|---|---|
| Segunda | Questão comentada | Texto + alternativas |
| Terça | Dica de estudo | Texto curto |
| Quarta | PDF / Resumo | Arquivo + descrição |
| Quinta | Questão comentada | Texto + alternativas |
| Sexta | Simulado mini (10Q) | Link ou PDF |
| Sábado | Motivação + streak | Texto |
| Domingo | Cronograma da semana | Imagem ou PDF |

### O que o aluno pensa

| Momento | Pensamento | Emoção |
|---|---|---|
| Entra no grupo | "Quantas pessoas... parece ativo" | Curiosidade |
| Vê mensagem boas-vindas | "Legal, me receberam" | Acolhimento |
| Recebe 1º conteúdo | "Questão comentada de verdade!" | Surpresa positiva |
| Dia 3 sem conteúdo | "Será que abandonaram?" | Desconfiança |
| Semana 2 consistente | "Posso confiar nisso" | Confiança |

### Pontos de dor

| Dor | Descrição | Severidade |

| Spam / off-topic | Membros poluem o grupo | Alta |
| Conteúdo genérico | PDFs sem curadoria | Média |
| Sem interação | Ninguém responde dúvidas | Alta |
| Muitas mensagens | Notificações excessivas | Média |

### Momentos de encantamento

| Momento | O que encanta |
|---|---|

| Questão com comentário detalhado | Qualidade acima do esperado |
| Cronograma visual da semana | "Agora sei o que estudar" |
| Resposta rápida a dúvida | Comunidade viva |
| Simulado no formato OAB | "Isso é preparação de verdade" |

### Oportunidades

| # | Oportunidade | Impacto |
|---|---|---|
| O9 | Bot de boas-vindas automatizado | Alto |
| O10 | Enquete semanal ("Qual matéria mais difícil?") | Médio |

### Métricas

| Métrica | Meta | Alarme |

| Retenção 30 dias | ≥ 40% | < 25% |
| Conteúdos/semana | ≥ 5 | < 3 |
